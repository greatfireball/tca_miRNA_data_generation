# NAME
clip_mapper.pl
# VERSION
0.9
# DEPENDENCIES
none
# DESCRIPTION
Adds the XM_IDs to the BED #4 column that were hit by the bed region
# PARAMETERS
none
# INPUT
- `.bed` file
- `species_A.gff`
# OUTPUT
`.bed` file with transcriptome assignment in column #4 at the end
- `;XM_...1,XM_...2`
