# NAME
071_filter_fastq_N.pl
# VERSION
0.9
# DEPENDENCIES
none
# DESCRIPTION
If the read contains N's, then it is discarded. Otherwise miraligner would consider N's as SNPs.
# PARAMETERS
none
# INPUT
FASTQ files. 
# OUTPUT
FASTQ files without any N's in their sequence.
